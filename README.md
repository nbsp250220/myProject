## 一、简介 {#1}
有时，桌面上的图标会变得杂乱，让人难以寻找文件、文件夹或应用程序。通过使用桌面图标整理工具，你可以轻松清理多余图标并进行分类，以便后续查找。该软件允许你将不同区域标记为不同分组，根据类别放置图标，并将多余图标放入可隐藏的文件夹中，节省空间，维护桌面的整洁。  
<br>
*支持平台：Windows7、Windows10、Windows11 (x86、x64)*

# 更多 {#016}

## 数据备份还原 {#16}
在全局设置处>数据处，你能看到默认开启了自动备份选项。  
备份数据的路径：目录下的<mark>/backup</mark>目录  
已知菜单的启动数据在<mark>/bin</mark>目录下，只需要将<mark>/backup</mark>下距离当天最近的数据<mark>(.db文件)</mark>复制覆盖到<mark>/bin</mark>目录下重启菜单即可。    
![菜单数据](https://gcore.jsdelivr.net/gh/nbsp250220/img/%E6%95%B0%E6%8D%AE%E6%BA%90.png)
